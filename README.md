# API

- Get all users, profile and history : GET <code>/v2/users</code>
- Create a user : POST <code>/v2/users</code>
- Edit a user : PUT <code>/v2/users/edit/:id</code>
- Delete a user : DELETE <code>/v2/users/delete/:id</code>
